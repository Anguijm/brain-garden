# Connections: threads across the garden

Most of this garden is filed by subject (games, AI, pets, cooking, and so on). But the
more interesting structure runs *sideways*: the same handful of ideas keep showing up
in notes that look unrelated. These pages are "maps of content," each one a single
thread pulled through several domains, so you can wander the garden by *idea* instead of
by topic and, ideally, get surprised.

If you would rather explore visually, the **graph** view (and the backlinks at the foot
of each note) shows the same links as a web you can click through.

## The threads

- **[Using AI well, without fooling yourself](using-ai-well)** — the garden's real
  spine. The same discipline (a fluent model is an assistant, not an oracle; make it
  expose its own downside) shows up in the AI engineering wing, the defense
  business-development playbook, and, surprisingly, a tree-frog terrarium build.
- **[Making with the Bambu X1C](making-with-the-x1c)** — one printer, four very
  different jobs, sometimes pulling in opposite directions (heavy game pieces versus
  ultralight airframes) on the same machine.
- **[Finite resources, ruthless economy](finite-resources)** — tokens, megabytes,
  grams, and person-hours: four hard ceilings, one shared move, do not ask for more
  headroom, get ruthless about what you spend it on.

## How to find more

Every note also carries cross-cutting **theme tags** you can browse: `verification`,
`using-ai-well`, `finite-resources`, `constraint-solving`, `feedback-loops`, and
`maker-toolchain`. Following a theme tag, or a note's backlinks, is the fastest way to
stumble onto a connection these three pages have not spelled out.
